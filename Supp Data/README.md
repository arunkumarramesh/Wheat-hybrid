**File Descriptions**

adaptors_novogene.fa - Adaptor sequences used for RNA-seq

SingleCopyOrthologues_matrix.tsv - 1:1 orthologs identified between Chinese Spring IWGSC RefSeq v2.1 and Paragon GCA949126075v1.

CSvP all genes.csv - Differentially Expressed genes between Chinese Spring and Paragon using IWGSC RefSeq v2.1 reference

CSvP all genes_PAR.csv - Differentially Expressed genes between Chinese Spring and Paragon using GCA949126075v1 reference

CS_PvCSxP all genes.csv - Differentially Expressed genes between Parents and Bybrids using IWGSC RefSeq v2.1 reference

CS_PvCSxP all genes_PAR.csv - Differentially Expressed genes between Parents and Bybrids using GCA949126075v1 reference

all_res_CS.csv - MBASED test for allele specific expression compared to median CS allele frequency across all sites using IWGSC RefSeq v2.1 reference

all_res_CS_0.5.csv - MBASED test for allele specific expression compared to 50:50 expected reference and alternate allele frequencies using IWGSC RefSeq v2.1 reference

all_res_PAR.csv - MBASED test for allele specific expression compared to median CS allele frequency across all sites using GCA949126075v1 reference
